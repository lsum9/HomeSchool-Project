# HomeSchool-Project

1. 목표 : 고등학생을 대상으로 하여 학생의 자율적 학습을 촉진하기 위한 선생님 학생의 상호작용 플랫폼 개발

2. 주요 기능
     - 회원(선생님, 학생) 관리
     - 시험 관리
     - 문항 관리
     - 강의 개설 및 관리
     - 학습자료 업로드
     - 시험 출제
     - 학생 수행에 대한 피드백 제공
     - 수강신청
     - 시험 응시
     - 영역별 성적 추이 확인

3. 개발환경 : jdk17, MariaDB, tomcat 10, SpringBoot 3.2.2, Gradle, jsp, thymeleaf
